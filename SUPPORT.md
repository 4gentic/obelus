# Support

Questions about how Obelus works, how it should work, or how to use it with a particular paper belong in [GitHub Discussions](https://github.com/obelus/obelus/discussions). That is also the right place to propose a direction before opening a pull request.

Bugs — anything that looks like broken behaviour, a missed invariant, or a regression — belong in [Issues](https://github.com/obelus/obelus/issues). Include the steps that reproduce the problem and your environment (browser or desktop build, OS, Obelus version).

Security reports are private. Send them to `security@obelus.app` or follow the process in [SECURITY.md](SECURITY.md). Do not file a public issue for a vulnerability.
