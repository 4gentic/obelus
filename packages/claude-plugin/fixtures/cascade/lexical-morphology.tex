\section{Introduction}

Agentic AI systems are being deployed in contexts where a single wrong action has irreversible consequences.
Every organisation deploying agents today chooses among three failure modes, each of which is unacceptable on its own terms.
Dangerous autonomy: the agent acts without judgement. Nagging escalation: the agent defers on everything. Hard-coded rules: an ex-ante decision tree that is stale the moment it meets conditions its author did not anticipate.
These three failure modes have the same root cause: each treats the boundary between autonomous action and human oversight as a configuration written once and treated as permanent.
The industry's response to the first two failures is an exhaustive decision tree, and it exhibits a distinct failure mode of its own.
